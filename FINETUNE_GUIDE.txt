阿里云百炼平台 - 微调操作指南

【当前数据文件】

格式: 百炼SFT标准JSONL格式

============================================
【详细步骤】
============================================

步骤一：获取正确的API Key
------------------------
1. 登录阿里云: https://console.aliyun.com/
2. 进入百炼平台: https://bailian.console.aliyun.com/
3. 左侧菜单点击「API-KEY管理」
4. 点击「创建 API Key」
5. 复制生成的Key (格式: sk-xxxxx)
6. 确保账户有足够余额（微调需要付费）

步骤二：上传数据集
------------------
1. 百炼平台左侧点击「数据管理」->「我的数据集」
2. 点击「创建数据集」
3. 填写信息:
   - 名称: wechat_chat_style
   - 类型: 选择「SFT监督微调」
   - 数据格式: 选择「对话数据」
4. 上传文件:
   - 选择本地文件
   - 上传: D:\code\AiWechat\data\processed\train_data.jsonl
5. 点击「创建」
6. 等待数据验证完成（约1-5分钟）

步骤三：创建微调任务
--------------------
1. 左侧点击「模型微调」->「训练任务」
2. 点击「创建训练任务」
3. 选择基础模型:
   - 推荐: qwen-turbo (速度快，成本低)
   - 或 qwen-plus (效果更好)
4. 选择数据集: 刚创建的数据集
5. 配置训练参数:
   - 训练轮数(epoch): 3 (推荐)
   - 学习率: 默认
   - 批次大小: 默认
6. 填写任务名称: wechat_style_clone
7. 点击「提交训练」

步骤四：等待训练完成
--------------------
- 训练时间: 约30分钟-2小时（取决于数据量）
- 可在任务列表查看进度
- 状态会显示: 待训练 -> 训练中 -> 训练完成

步骤五：部署模型
----------------
1. 训练完成后，点击任务进入详情
2. 点击「部署模型」
3. 选择部署方式:
   - 专用部署: 独享资源，响应快
   - 公共部署: 共享资源，成本低
4. 等待部署完成
5. 记录模型ID (格式如: qwen-turbo-finetuned-xxx)

步骤六：配置项目
----------------
1. 编辑 D:\code\AiWechat\.env 文件
2. 填写:
   DASHSCOPE_API_KEY=你的正确Key
   DASHSCOPE_FINETUNED_MODEL=微调后的模型ID
3. 保存文件

============================================
【常见问题】
============================================
Q: 数据验证失败？
A: 检查JSONL格式是否正确，每行必须是完整JSON

Q: 训练失败？
A: 可能是数据量太少或格式问题，建议至少500条

Q: 模型效果不好？
A: 可以增加训练轮数，或使用更大的基础模型
